Hello! Welcome to this Dog breeds website!


This website uses the dog API to provide information on various dog breeds

Info

When you're ready, click the "Breeds" tab in the navigation bar at the top of the page in order to move to a page that has a list of dog breeds. From there you can click on any of the breeds to view more info on them. Alternatively, if you already have the id of a breed you want too see you can enter it after typing a / after breeds in navigation bar. You can also get some breed facts by clicking on the "Facts" tab in the navigation bar.
